遇到问题，及解决方法
1、stm32l151c8t6串口首字符乱码
解决方法：端口复用语句需要放在配置前面
GPIO_PinAFConfig(GPIOA,GPIO_PinSource9,GPIO_AF_USART1);
GPIO_PinAFConfig(GPIOA,GPIO_PinSource10,GPIO_AF_USART1);

2、stm32l151c8t6串口配置成浮空输入进入不了中断服务函数
解决方法：stm32l151c8t6与stm32f103系列还是有差别，RX引脚不要配置成浮空输入，要配置复用模式
GPIO_InitStruct.GPIO_Mode=GPIO_Mode_AF;

3、stm32l151c8t6外部中断配置普通GPIO浮空输入导致不稳态出现，中断引脚电平有波动导致误触发外部中断
解决方法：当引脚外部没接上拉电阻，GPIO内部配置成上拉输入

4、stm32l151c8t6外部中断与LPCD功能结合时输出电平配置注意事项
根据复旦微550用户手册，若配置MCU中断引脚为内部上拉输入，则550IRQ引脚跟MCU的一致。
